Encurtador de Links



Para utilizar o encurtador de links, siga estes passos:

  1. Cole o link que deseja encurtar no campo acima.
  2. Clique no botão "Encurtar".
  3. O link encurtado será exibido abaixo. Clique em "Copiar link" para copiá-lo para a área de transferência.
